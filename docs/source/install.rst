Installing SpaceTime GIS
------------------------

.. currentmodule:: spacetimegis

You can get the library directly from PyPI:

.. code-block:: console

    > pip install spacetimegis
    > spacetimegis init 
    > spacetimegis db upgrade
    > spacetimegis worker 
    > spacetimegis runserver

Before installing spacetimegis on Windows it is recommended to install the following packages

.. code-block:: console

    gdal
    rasterio
    shapely
    fiona

.. note:: `SpaceTime GIS` requires Python 3 or later.
