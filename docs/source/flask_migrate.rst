Installing Flask_Migrate
------------------------

.. currentmodule:: Flask_Migrate

You can get the library directly from PyPI:

.. code-block:: console

    > pip install Flask_Migrate

.. code-block:: console

    spacetimegis db init
    spacetimegis db migrate
    spacetimegis db upgrade
    spacetimegis db --help

.. note:: `SpaceTime GIS` requires Python 3 or later.
